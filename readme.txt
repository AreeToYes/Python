# Курсова работа: Система за Управление на Автосалон
Курс: Програмиране на Python (2025/2026)

## 1. Описание на проекта
Този проект представлява конзолно приложение за симулиране на автосалон. Целта на разработката е да демонстрира прилагането на Обектно-ориентирано програмиране (ООП) на Python, работа с класове, методи, цикли и условни конструкции.

## 2. Структура на проекта
Проектът е разделен на логически компоненти в съответствие с изискванията:
* models.py - Съдържа дефиницията на основния клас `Car` (Автомобил), неговия конструктор (__init__) и два специфични метода за обработка на данните.
* main.py - Главният изпълним файл, в който се инициализират обектите, създава се списък от автомобили и се демонстрира филтриране и извеждане на информацията чрез цикли и логически проверки.
* requirements.txt - Описание на необходимите софтуерни зависимости (проектът разчита изцяло на вградената библиотека на Python).

## 3. Функционалности
* Създаване на динамични обекти от тип автомобил с характеристики: марка, модел и конски сили.
* Автоматично генериране на четимо форматирано име на превозното средство чрез метод в класа.
* Логическа проверка и филтриране на автомобилите по критерий мощност (над 150 к.с.).

## 4. Как се стартира проектът
1. Уверете се, че имате инсталиран Python (версия 3.8 или по-нова).
2. Отворете проекта в PyCharm или предпочитана среда за разработка.
3. Стартирайте главния файл чрез командата в терминала:
   python main.py
